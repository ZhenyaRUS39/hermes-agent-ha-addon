# Hermes Agent Home Assistant Addon

AI-agent with advanced tool-calling, integrated with Home Assistant via HA API.

## Features

- Native Home Assistant API integration
- Voice control via Marusia/Yandex
- Smart home device management
- Flexible tool system

## Installation

1. Add this repository to Home Assistant Addon Store
2. Install "Hermes Agent" addon
3. Provide Home Assistant API token
4. Start the addon

## Configuration

- `ha_token` - Home Assistant Long-lived access token
- `log_level` - logging level
- `update_on_start` - update Hermes on startup
